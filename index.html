CHEATSHEET CSS

Les sélecteurs :

  • .class :

    Permet de sélectionner un élement par rapport à sa classe

  • #id :

    Permet de sélectionner un élement par rapport à sa classe

  • * :

    Sélectionne tout les élements

  • div :

    Sélectionne tous les élements de la div

  • div, p :

    Sélectionne toutes les div et tous les p

  • div > p :

    Sélectionne tout les p qui sont imbriqués dans la div

  • div + p :

    Sélectionne tout les élements p qui sont placés directement après la div

  • div ~ p :

    Sélectionne tout les élements p après la div mentionnée

Les pseudo-classes

  • a:link :

    Sélectionne tous les liens non visités

  • a:visited :

    Sélectionne tous les liens qui ont été visités

  • a:hover :

    Sélectionne les liens que le curseur de la souris survole

  • p::before :

    Insère du contenu avant l'élément p

  • p::after :

    Insère du contenu après l'élément p

  • p:first-child :

    Sélectionne tous les éléments p qui sont les premiers du conteneur

  • p:last-child :

    Sélectionne tous les éléments p qui sont les derniers du conteneur

  • p:nth-child(X) :

    Sélectionne le Xième enfant du conteneur
